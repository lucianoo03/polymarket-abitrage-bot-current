## 🛠 Troubleshooting Tips

If you run into common issues, try these solutions:

- **Bot does not start:**  
  Make sure you have extracted all files correctly. Try running the `.exe` file again as Administrator (right-click > Run as Administrator).  

- **Bot cannot connect to Polymarket:**  
  Check your internet connection. Ensure Polymarket’s website is accessible from your browser.

- **Trades do not execute:**  
  Double-check your account details in the settings. Verify your Polymarket account is active and has funds or tokens to trade.

- **Errors in the program:**  
  Close the bot, and open the folder. Look for a file called `error.log` or similar to see the problem details. Share this file with someone who can help if needed.
